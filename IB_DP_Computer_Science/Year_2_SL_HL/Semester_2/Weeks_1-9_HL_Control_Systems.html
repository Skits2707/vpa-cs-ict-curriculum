Topic 7.2: Distributed Systems 分布式系统 Fēnbùshì Xìtǒng

Week 3 — Distributed Control Fundamentals 分布式控制基础

LessonContentActivities
1Distributed systems: multiple autonomous nodes cooperating; no single point of control (分布式系统, 节点, 自治)Compare to centralized using Venn diagram
2Communication in distributed systems: message passing, shared memory, RPC, publish-subscribeDiagram each pattern; match to use case
3Consensus problems: agreement, validity, termination; Byzantine generals problem (conceptual)Simulation: reaching consensus with faulty nodes
4Distributed algorithms: leader election, mutual exclusion (Ricart-Agrawala conceptual)Trace leader election on small network
5TOK: In a distributed system, no single node has complete knowledge. Can the system as a whole be said to "know" things that no individual node knows?Discussion; connection to emergent properties
TOK Knowledge Question: Distributed systems exhibit emergent behaviour that no single node is aware of. Is this emergence a form of collective knowledge, or merely the appearance of knowledge? Can a system "know" something without any part of it knowing it?
