Introduction

VPA students enter IB DP Computer Science with strong foundations from IGCSE 0478. This guide maps the transition, identifies knowledge gaps, and provides targeted preparation to ensure every student starts Year 1 confident and ready.

2. Knowledge Gaps to Address

Gap 1: Systems Thinking (Topic 1)

IGCSE focuses on technical components. IB DP Topic 1 requires organisational understanding: stakeholders, feasibility, changeover, project management. Students must shift from "how does this work?" to "how do we implement this in an organisation?"

Preparation Tasks:
  • Interview a staff member about a system change at VPA
  • Read a case study of a failed IT implementation (e.g., NHS IT project)
  • Define: stakeholder, feasibility, milestone, changeover in your own words

Gap 2: Binary Depth (Topic 2.4)

IGCSE covers binary, hex, and basic ASCII. IB DP adds:

  • Two's complement for negative numbers
  • Floating-point representation (mantissa and exponent)
  • Truncation and rounding errors
  • Binary shifts (logical and arithmetic)

Gap 3: Logic Gates (Topic 2.5)

IGCSE covers AND, OR, NOT. IB DP requires:

  • NAND, NOR, XOR, XNOR
  • Constructing circuits from Boolean expressions
  • Deriving expressions from truth tables
  • Simple applications (half-adders, security systems)

Gap 4: Algorithm Rigor (Topic 4)

IGCSE programming is practical. IB DP requires:

  • Standard IB pseudocode (not Python-specific)
  • Trace tables for all algorithms
  • Efficiency analysis (Big O at HL)
  • Formal testing strategies
