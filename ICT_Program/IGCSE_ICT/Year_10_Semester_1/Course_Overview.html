Semester Objectives

By the end of Semester 1, students will be able to:

  • ☐ Perform advanced data analysis using spreadsheets and databases
  • ☐ Create multi-page websites using HTML and CSS
  • ☐ Design and query relational databases
  • ☐ Integrate data between applications (web forms, database links)
  • ☐ Work with large datasets efficiently
  • ☐ Test and debug web pages across browsers

Paper 3 Exam Format

  • Duration: 2 hours 30 minutes
  • Marks: 80
  • Structure: Integrated tasks combining data analysis, web authoring, and databases
  • Evidence: Screen shots, printed pages, electronic files in designated folders
  • Key difference from Paper 2: Paper 3 requires INTEGRATION between applications (e.g., web form writes to database, spreadsheet data displayed on web page)
