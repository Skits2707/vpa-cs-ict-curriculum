3.3 Sample Questions

Question 1 (2 marks) — State two differences between RAM and ROM.

Mark Scheme:
• RAM is volatile / ROM is non-volatile (1 mark)
• RAM can be read from and written to / ROM is read-only (1 mark)
• RAM is used for currently running programs / ROM stores BIOS/boot instructions (1 mark)
Any two valid differences accepted.

Question 2 (4 marks) — Describe the advantages and disadvantages of using a wireless network in a school.

Mark Scheme:
Advantages (up to 2 marks):
• No cables needed — easier to set up and move devices (1)
• Devices can connect from anywhere in the school (1)
• Cheaper infrastructure (no cabling costs) (1)
Disadvantages (up to 2 marks):
• Slower speeds than wired networks (1)
• Security risks — easier to intercept (1)
• Interference from other devices (1)
• Limited range — may need repeaters (1)

Question 3 (6 marks) — A company wants to implement a new database system. Explain the stages of systems analysis and design they should follow.

Mark Scheme (6 marks):
• Analysis — identify problem, gather requirements, investigate current system (1)
• Design — plan solution, create data structures, design input/output screens (1)
• Development / Implementation — build database, create forms, write queries (1)
• Testing — test with sample data, check for errors, user acceptance testing (1)
• Installation / Deployment — install on user computers, train staff (1)
• Documentation — user guide, technical guide, training materials (1)
• Maintenance — fix bugs, add features, update as needed (1)
• Evaluation — check against original requirements, identify improvements (1)
Any 6 stages with explanation = 6 marks

4.3 Sample Task — Document Production (30 marks)

Scenario: A school is preparing a newsletter for parents. You need to create a professional document using the provided source files.

Requirements:

  1. Create a newsletter with appropriate headings, subheadings, and body text (5 marks)
  2. Apply consistent formatting using styles (5 marks)
  3. Insert and format a table showing term dates (5 marks)
  4. Import and position images with appropriate sizing and wrapping (5 marks)
  5. Add page numbers, headers, and footers (5 marks)
  6. Perform a mail merge to create personalised letters for 10 parents (5 marks)

4.4 Sample Task — Data Manipulation (30 marks)

Scenario: A sports shop has provided sales data. You need to analyse the data using a spreadsheet.

Requirements:

  1. Import data from a CSV file and format appropriately (5 marks)
  2. Create formulas to calculate totals, averages, and percentages (5 marks)
  3. Use functions (SUM, AVERAGE, MAX, MIN, COUNTIF) (5 marks)
  4. Create a chart showing sales trends (5 marks)
  5. Sort and filter data to answer specific questions (5 marks)
  6. Apply conditional formatting to highlight key data (5 marks)

4.5 Sample Task — Presentations (20 marks)

Scenario: Create a 5-slide presentation about healthy eating for a Year 6 audience.

Requirements:

  1. Design a consistent slide master with appropriate colours and fonts (5 marks)
  2. Create 5 slides with relevant content and images (5 marks)
  3. Add appropriate transitions and animations (5 marks)
  4. Ensure the presentation is suitable for the target audience (5 marks)

9. EXAM TECHNIQUE GUIDE

Paper 1 — Theory

  • Time management: 100 marks in 120 minutes = 1.2 minutes per mark
  • Read the command word carefully: State (1 line), Describe (how it works), Explain (reasons why), Evaluate (advantages AND disadvantages)
  • Show all working for calculation questions
  • Use diagrams where appropriate — they can earn marks even if text is unclear
  • Answer all questions — no negative marking, so guess if unsure

Paper 2 & 3 — Practical

  • Read all tasks before starting — plan your time
  • Save your work frequently — every 10 minutes
  • Check file formats — save in the required format
  • Test everything — check links, formulas, print preview
  • Leave 10 minutes at the end for checking and saving
