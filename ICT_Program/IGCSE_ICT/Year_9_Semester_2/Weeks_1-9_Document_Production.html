Weeks 1–9: Document Production

Syllabus: CIE IGCSE ICT (0417) Topic 10 | Software: Microsoft Word / LibreOffice Writer

Week 1 — Document Setup & Formatting

Syllabus: 10.1 Document Production — Page Setup, Styles, Formatting
Keywords: style 样式 (yàngshì), template 模板 (múbǎn), margin 页边距 (yèbiān jù), orientation 方向 (fāngxiàng), header 页眉 (yèméi), footer 页脚 (yèjiǎo), page break 分页符 (fēnyè fú), section break 分节符 (fēnjié fú)
Activity 1 (20 min): "Exam Setup" — Create a document with A4, 2.5cm margins, portrait, page numbers in footer, name in header.

Activity 2 (15 min): Style application — Apply Heading 1, Heading 2, and Body Text to a 3-page document. Generate Table of Contents.

Activity 3 (5 min): Section break practice — Change page 2 to landscape while keeping page 1 portrait.
Exam Tip: Always set up page layout FIRST (margins, orientation, headers/footers) before adding content. This saves time and prevents reformatting.

Week 2 — Tables & Lists

Syllabus: 10.2 Tables, Bullets, Numbering
Keywords: table 表格 (biǎogé), cell 单元格 (dānyuán gé), merge 合并 (hébìng), split 拆分 (chāifēn), border 边框 (biānkuàng), shading 底纹 (dǐwén), bullet 项目符号 (xiàngmù fúhào), numbering 编号 (biānhào)
Activity 1 (15 min): Create a complex table: timetable with merged cells for lunch, colored headers, specific border styles.

Activity 2 (15 min): Convert text to table and vice versa. Practice with class list data.

Activity 3 (10 min): Multi-level numbering — Create a numbered list with 3 levels (1, 1.1, 1.1.1).

Week 3 — Images, Text Wrapping & Captions

Syllabus: 10.3 Graphics and Images
Keywords: text wrapping 文字环绕 (wénzì huánrào), inline 嵌入 (qiànrù), square 四周型 (sìzhōu xíng), tight 紧密型 (jǐnmì xíng), behind text 衬于文字下方 (chèn yú wénzì xiàfāng), caption 题注 (tízhù), crop 裁剪 (cáijiǎn), resize 调整大小 (tiáozhěng dàxiǎo)
Activity 1 (15 min): Insert image and apply 4 different wrapping styles. Observe differences.

Activity 2 (15 min): "Professional Document" — Insert 3 images, crop one to focus on detail, resize to consistent width, add captions.

Activity 3 (10 min): Troubleshooting — "My image keeps moving!" Solution: anchor to page, use square wrapping, lock position.

Week 4 — Mail Merge

Syllabus: 10.4 Mail Merge
Keywords: mail merge 邮件合并 (yóujiàn hébìng), data source 数据源 (shùjù yuán), recipient list 收件人列表 (shōujiàn rén lièbiǎo), merge field 合并域 (hébìng yù), primary merge document 主文档 (zhǔ wéndàng), filter 筛选 (shāixuǎn)
Activity 1 (20 min): Create a certificate mail merge. Data source: CSV with Name, Class, Achievement. Document: certificate template.

Activity 2 (15 min): Filtered merge — Only merge for students in Class 9A.

Activity 3 (5 min): "What if the merge goes wrong?" Common errors: wrong data source, field names with spaces, missing fields.
Exam Tip: Mail merge is always worth 6–10 marks in Paper 2. Steps: 1) Select recipients (CSV/database), 2) Insert merge fields, 3) Preview results, 4) Complete merge. Save ALL output documents.

Week 5 — Advanced Formatting: Columns, Borders, Tabs

Activity 1 (15 min): Newsletter format — 2 columns, drop cap, horizontal line separator.

Activity 2 (15 min): Tab stops — Create a price list with decimal-aligned tabs.

Activity 3 (10 min): Page border and watermark — "Draft" watermark, decorative border.

Week 6 — Proofreading & Document Review

Syllabus: 10.5 Proofreading
Keywords: proofread 校对 (jiàoduì), spell check 拼写检查 (pīnxiě jiǎnchá), grammar check 语法检查 (yǔfǎ jiǎnchá), thesaurus 同义词词典 (tóngyìcí cídiǎn), find/replace 查找替换 (cházhǎo tìhuàn), word count 字数统计 (zìshù tǒngjì)
Activity 1 (15 min): Error-finding race — Document with 15 deliberate errors. Find and fix ALL.

Activity 2 (15 min): Find and replace — Change all instances of "UK" to "United Kingdom" using formatting (bold).

Activity 3 (10 min): Thesaurus use — Replace 5 overused words with more precise alternatives.

Week 7 — Timed Practice 1

40-Minute Task:
  1. Set up document: A4, margins 2cm, header with name, footer with page numbers
  2. Type provided text (300 words) with Heading 1 and Body Text styles
  3. Insert a 4x4 table with formatted headers and borders
  4. Insert 2 images with appropriate wrapping and captions
  5. Apply 2-column formatting to one section
  6. Proofread and correct 3 deliberate errors

Week 8 — Timed Practice 2

40-Minute Task:
  1. Import data from CSV file
  2. Create mail merge letter with 5 merge fields
  3. Filter recipients by city
  4. Complete merge and save as individual documents
  5. Create envelope layout with merge fields

Week 9 — Document Production Assessment

Mock Paper 2 Section A (40 min):

Students complete a timed document production task from a past paper (MJ 2022 or ON 2021). Teacher marks against official mark scheme.

Marking Focus:

  • Page setup and layout (5 marks)
  • Text formatting and styles (8 marks)
  • Tables and lists (6 marks)
  • Images and graphics (4 marks)
  • Mail merge (10 marks)
  • Proofreading and accuracy (7 marks)

Common Errors to Avoid

  • ❌ Forgetting to save to the correct folder with correct filename
  • ❌ Using manual formatting instead of styles
  • ❌ Not previewing mail merge before completing
  • ❌ Missing page numbers or headers
  • ❌ Not checking spell check after completing
  • ❌ Cropping images instead of resizing (loss of quality)
  • ❌ Using "float over text" when square wrapping is requested
