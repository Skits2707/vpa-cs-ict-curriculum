VPA ICT Program — Grades 4-5: Digital Fluency

Academic Year: 2024–2025 | Total Weeks: 36 | Lesson Duration: 40 minutes

Target Group: EAL learners (Mandarin L1), mixed ability | Standards: CSTA K-12, ISTE Standards

Typing Target: 20 WPM (words per minute) by end of Grade 5 with 90% accuracy

Program Vision

Grades 4–5 build on digital foundations by developing fluency in productivity tools (word processing, spreadsheets, presentations), responsible internet use, and introductory programming with Scratch. Students begin creating polished digital products for real audiences while developing critical thinking about online information.

Assessment Overview

Formative (ongoing):

  • Weekly typing speed checks (WPM tracking chart)
  • Document / spreadsheet / presentation checkpoints
  • Scratch project milestones (movement → loops → variables)
  • Peer code reviews with structured feedback forms

Summative (term-end):

  • Term 1: Timed document production task (40 min)
  • Term 2: Data analysis + presentation project
  • Term 3: Research report + Scratch game project

Bilingual Support

Sentence Frames for Writing:

  • "The data shows that..." / "数据显示... (shùjù xiǎnshì...)"
  • "According to my research..." / "根据我的研究... (gēnjù wǒ de yánjiū...)"
  • "I recommend... because..." / "我建议...因为... (wǒ jiànyì... yīnwèi...)"
