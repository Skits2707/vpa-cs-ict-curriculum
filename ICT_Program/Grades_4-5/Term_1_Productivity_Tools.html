Term 1: Productivity Tools — Weeks 1–12

Theme: Advanced word processing, formatting, images, tables | Duration: 40 minutes per lesson

Term Overview

Students develop professional document creation skills using Microsoft Word or Google Docs. Building on Grades 2–3 foundations, they learn advanced formatting, table creation, image integration, headers/footers, and collaborative editing. The term culminates in a formal document production assessment.

Week-by-Week Scheme of Work

Week 1 — Typing Review & Baseline Assessment

Keywords: WPM (words per minute) 每分钟字数 (měi fēnzhōng zìshù), accuracy 准确率 (zhǔnquè lǜ), home row 基准键行 (jīzhǔn jiàn háng), posture 姿势 (zīshì), touch typing 盲打 (mángdǎ)
Activity 1 (10 min): Baseline typing test on TypingClub or typing.com. Record WPM and accuracy.

Activity 2 (15 min): Goal-setting worksheet. Students write: "My current WPM is ___. By the end of Grade 5, I want to reach ___ WPM."

Activity 3 (15 min): Posture and ergonomics lesson. "Feet flat, back straight, fingers curved, eyes on screen!"
Baseline Data: Record WPM and accuracy for every student. Track weekly.

Week 2 — Touch Typing Practice

Activity 1 (20 min): TypingClub guided lessons (Levels 1–3). Focus on accuracy over speed.

Activity 2 (15 min): "Finger Gym" — Exercises for each finger: pinky practice, ring finger drills, thumb spacebar.

Activity 3 (5 min): WPM check — Did speed improve from last week?

Week 3 — Speed & Accuracy Challenges

Activity 1 (15 min): Typing tournament — Students race in pairs on the same passage. Winner advances.

Activity 2 (15 min): "Accuracy Challenge" — Type a passage perfectly. One mistake = start over! Builds careful typing.

Activity 3 (10 min): WPM graph update — Students plot their weekly WPM on a personal chart.

Week 4 — Advanced Formatting

Keywords: font 字体 (zìtǐ), alignment 对齐 (duìqí), bold 粗体 (cūtǐ), italic 斜体 (xiétǐ), underline 下划线 (xiàhuàxiàn), line spacing 行距 (hángjù), margins 页边距 (yèbiān jù)
Activity 1 (15 min): "Format Detective" — Teacher displays a formatted document. Students must recreate the exact formatting.

Activity 2 (15 min): "Professional Document Challenge" — Students format a messy, unformatted letter into a professional-looking document with proper alignment, spacing, and font choices.

Activity 3 (10 min): Font choice discussion — "When is Comic Sans OK? When should we use Times New Roman?"
Formative: Document matches teacher model within 90% accuracy.

Week 5 — Tables in Documents

Keywords: table 表格 (biǎogé), row 行 (háng), column 列 (liè), cell 单元格 (dānyuán gé), border 边框 (biānkuàng), merge 合并 (hébìng)
Activity 1 (15 min): "Class Timetable" — Students create a weekly timetable with 5 columns (Mon–Fri) and 6 rows (periods).

Activity 2 (15 min): Table formatting — Change border colors, add shading to header row, merge cells for lunch break.

Activity 3 (10 min): "Table vs. Text" discussion — When is a table better than a list?
Formative: Table has correct rows/columns, formatted header, merged cells.

Week 6 — Images & Text Wrapping

Keywords: insert image 插入图片 (chārù túpiàn), text wrapping 文字环绕 (wénzì huánrào), inline 嵌入 (qiànrù), square 四周型 (sìzhōu xíng), behind text 衬于文字下方 (chèn yú wénzì xiàfāng)
Activity 1 (15 min): "Travel Brochure" — Students create a 1-page travel brochure about a country, with 3 pictures and wrapped text.

Activity 2 (15 min): Text wrapping experiment — Same picture, 3 different wrapping styles. Compare results.

Activity 3 (10 min): Image sizing and cropping — Students crop an image to focus on one detail.
Formative: Brochure has 3+ images with appropriate text wrapping.

Week 7 — Headers, Footers & Page Numbers

Keywords: header 页眉 (yèméi), footer 页脚 (yèjiǎo), page number 页码 (yèmǎ), page break 分页符 (fēnyè fú)
Activity 1 (15 min): "Formal Letter" — Students create a formal letter with their name in the header and page numbers in the footer.

Activity 2 (15 min): Multi-page document — Create a 3-page document where each page has a different header.

Activity 3 (10 min): "When do we need page breaks?" — Discussion and practice inserting manual page breaks.
Formative: Document has header, footer, and correct page numbering.

Week 8 — Review & Mini-Project

Activity 1 (30 min): "All About Me" Document — Students create a 2-page document about themselves with:
  • Title page with name and photo
  • Formatted text with bold/italic/color
  • A table showing favorites (food, sport, color, subject)
  • 2+ pictures with text wrapping
  • Page numbers and their name in the footer
Activity 2 (10 min): Peer review using checklist.

Week 9 — Document Collaboration

Keywords: collaborate 协作 (xiézuò), share 分享 (fēnxiǎng), comment 评论 (pínglùn), suggest 建议 (jiànyì), track changes 修订 (xiūdìng)
Activity 1 (15 min): Google Docs sharing — Teacher demonstrates sharing permissions. Students share a document with their partner.

Activity 2 (15 min): "Partner Editing" — Students write a paragraph, share with partner, partner adds 2 constructive comments.

Activity 3 (10 min): Class discussion — "What makes feedback helpful?" Create a class feedback anchor chart.
Formative: Student gives 2 specific, kind, and helpful comments.

Week 10 — Proofreading & Editing

Keywords: proofread 校对 (jiàoduì), edit 编辑 (biānjí), spelling 拼写 (pīnxiě), grammar 语法 (yǔfǎ), punctuation 标点 (biāodiǎn)
Activity 1 (15 min): "Error-Finding Challenge" — Teacher provides a document with 10 deliberate errors. Students race to find and fix them.

Activity 2 (15 min): Self-editing checklist — Students apply a checklist to their own work: capital letters? periods? spelling? spaces?

Activity 3 (10 min): Peer proofreading — Exchange documents, use highlighters: yellow = great!, pink = check this.
Formative: Finds and fixes 7/10 errors in the challenge document.

Week 11 — Real-World Writing

Activity 1 (20 min): "Letter to the Principal" — Students write a formal letter about a school improvement idea. Must include:
  • Proper letter format (date, address, greeting)
  • Clear purpose in opening paragraph
  • Supporting reasons with details
  • Polite closing
Activity 2 (15 min): Peer review and revision using comment feature.

Activity 3 (5 min): Share with teacher for feedback.

Week 12 — Term 1 Assessment

Timed Document Production (40 min):

Students receive a task sheet and must create a professional document from scratch:

  1. Type a provided text (150 words) with correct formatting (font, size, alignment)
  2. Insert and format a table with 3 columns and 4 rows
  3. Add 1 image with appropriate text wrapping
  4. Add a header with their name and a footer with page numbers
  5. Apply bold, italic, and underline to specific parts of the text

Marking Criteria:

  • Bronze: 3/5 elements completed correctly
  • Silver: 4/5 elements completed correctly
  • Gold: All 5 elements completed correctly + professional appearance
