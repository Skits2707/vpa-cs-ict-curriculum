IGCSE Computer Science (0478)

Cambridge Assessment International Education • Year 9 & Year 10
The Cambridge IGCSE Computer Science (0478) syllabus develops an understanding of computational thinking and programming skills. Students study topics including data representation, hardware and software, networks, and programming fundamentals using Python. Assessment is via two written papers and a practical programming assessment.

A-Level Computer Science (9618)

Cambridge International A-Level • AS (Year 11) & A2 (Year 12)
The Cambridge International A-Level Computer Science (9618) syllabus builds on IGCSE foundations with advanced theory and practical programming. AS Level covers information representation, hardware, software, security, and programming fundamentals. A2 Level extends into data communication, AI, advanced programming projects, and system architecture. Assessment includes written papers and a practical programming project.

ICT Program (Grades 2 → IGCSE ICT 0417)

Cambridge IGCSE ICT (0417) • Progressive ICT from Primary to IGCSE
The VPA ICT Program provides a progressive pathway from foundational digital literacy in primary grades through to IGCSE ICT (0417). Students develop practical skills in productivity software, digital creation, programming fundamentals, databases, web authoring, and cybersecurity. The program emphasizes hands-on application and real-world problem solving.

IB DP Computer Science (SL/HL)

International Baccalaureate Diploma Programme • Year 1 & Year 2
The IB DP Computer Science course (SL/HL) develops computational thinking skills through the study of system fundamentals, computer organization, networks, abstract data structures, resource management, and control systems. The course emphasizes the IB learner profile and includes an Internal Assessment (IA) programming project. HL students study additional advanced topics including abstract data structures, resource management, and control systems.
