IGCSE Computer Science — Year 9 Semester 1 Course Overview (Excerpt)

STUDENT PROFILE
• EAL (English as Additional Language): Heavy bilingual scaffolding
• Mixed Ability: Differentiated worksheets (core/extension)
• Prior Knowledge: Minimal computing background assumed
• Programming: Introduced lightly in Weeks 16-17

PROGRAMMING STRATEGY
Year 9: Theory-focused with light Python exposure
Year 10: Full programming curriculum
Rationale: 2 lessons/week insufficient for deep programming + theory

ASSESSMENT CALENDAR

Data Representation Test     | Week 6  | 30 min | End of unit
Hardware Quiz                | Week 10 | 15 min | In-class
End-of-Semester Mock         | Week 18 | 1 hour | Paper 1 style

STANDARD LESSON TEMPLATE (40 Minutes)

3-12 min  | Direct Instruction | New concept, visual, bilingual keywords
12-25 min | Guided Practice | Worked examples, pair work, AfL
25-35 min | Independent Practice | Worksheet, exam question
35-40 min | Plenary + Homework | Check understanding, set HW

Week-by-Week Scheme of Work — Sample Week (Excerpt)

WEEK 5: IMAGES (Pixels, Resolution, Colour Depth)

Lesson 1 (40 min): How Digital Images Work
• Starter (5 min): "Guess the resolution" — show 3 versions of same image
• Main (25 min):
  - Pixel = picture element (像素 xiangsu)
  - Resolution = width x height in pixels (分辨率 fenbianlv)
  - Colour depth = bits per pixel (颜色深度 yan se shen du)
  - File size formula: width x height x colour depth (bits)
  - Worked example: 800x600 at 24-bit colour
• Practice (10 min): Calculate file sizes for 3 scenarios
  BRONZE: 100x100 at 8-bit
  SILVER: 640x480 at 16-bit
  GOLD: 1920x1080 at 24-bit + convert to KB
• Plenary (5 min): "Why does a 4K video take so much space?"

Homework: Past paper question (images) + vocabulary flashcards
