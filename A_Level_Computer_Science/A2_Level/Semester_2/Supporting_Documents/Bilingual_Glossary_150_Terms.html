Unit 1: Data Representation
English中文PinyinDefinition
Binary二进制ÈrjìnzhìA number system using base 2, with digits 0 and 1.
Denary / Decimal十进制ShíjìnzhìA number system using base 10, with digits 0–9.
Hexadecimal十六进制Shíliù jìnzhìA number system using base 16, with digits 0–9 and A–F.
Bit位WèiThe smallest unit of data, representing 0 or 1.
Byte字节ZìjiéA group of 8 bits, the standard unit of storage.
Nibble半字节Bàn zìjiéA group of 4 bits, half of a byte.
Two's Complement补码BǔmǎA method for representing signed integers in binary, allowing subtraction by addition.
Sign and Magnitude符号幅度Fúhào fúdùA method for representing signed integers where the first bit is the sign and the rest represent the magnitude.
Overflow溢出YìchūAn error that occurs when a calculation produces a result too large to be stored.
Underflow下溢Xià yìAn error when a number is too small (close to zero) to be represented.
⭐ Floating-Point浮点数Fúdiǎn shùA representation of real numbers using a mantissa and an exponent, allowing a wide range of values.
⭐ Mantissa尾数WěishùThe fractional part of a floating-point number, representing the significant digits.
⭐ Exponent指数ZhǐshùThe power to which the base is raised in a floating-point number.
⭐ Normalisation规范化Guīfàn huàThe process of adjusting a floating-point number so the mantissa is within a standard range (e.g., 0.5 ≤ mantissa < 1).
ASCII美国信息交换标准代码Měiguó xìnxī jiāohuàn biāozhǔn dàimǎA 7-bit character encoding standard representing 128 characters.
Unicode统一码Tǒngyī mǎA universal character encoding standard supporting multiple writing systems, including Chinese.
Binary-Coded Decimal (BCD)二进制编码的十进制Èrjìnzhì biānmǎ de shíjìnzhìA representation where each decimal digit is stored as a 4-bit binary number.
Gray Code格雷码Géléi mǎA binary code where consecutive values differ by only one bit, reducing errors in digital systems.
Resolution分辨率Fēnbiàn lǜThe number of pixels per unit of length in an image, or the number of bits used to represent a sample in sound.
Colour Depth色深Sè shēnThe number of bits used to represent the colour of a single pixel.
Metadata元数据Yuán shùjùData that describes other data, such as image dimensions, file type, or creation date.
Sampling Rate采样率Cǎiyàng lǜThe number of samples taken per second when digitising sound.
Sampling Resolution采样分辨率Cǎiyàng fēnbiàn lǜThe number of bits used to represent each sample of sound.
⭐ Nyquist Theorem奈奎斯特定理Nài kuí sī tè dìnglǐStates that the sampling rate must be at least twice the highest frequency in the signal to avoid loss.
Bit Rate比特率Bǐtè lǜThe number of bits processed per unit of time, typically bits per second (bps).
Lossless Compression无损压缩Wúsǔn yāsuōCompression that reduces file size without losing any data, allowing perfect reconstruction.
Lossy Compression有损压缩Yǒusǔn yāsuōCompression that reduces file size by permanently removing some data, sacrificing quality.
Run-Length Encoding (RLE)行程长度编码Xíngchéng chángdù biānmǎA simple lossless compression method replacing repeated values with a count and value.

Unit 3: Hardware
English中文PinyinDefinition
Von Neumann Architecture冯·诺依曼架构Féng nuò yī màn jiàgòuA computer design where instructions and data are stored in the same memory and processed sequentially.

⭐ Segmentation分段Fēn duànA memory management scheme that divides memory into variable-sized segments based on logical divisions.
⭐ Virtual Memory虚拟内存Xūnǐ nèicúnA technique that uses secondary storage to extend the apparent capacity of main memory.

Output Device输出设备Shūchū shèbèiA hardware device used to present data from a computer system to the user.

HDD机械硬盘Jīxiè yìngpánA storage device using spinning magnetic platters to store data.
RAID独立磁盘冗余阵列Dúlì cípán rǒngyú zhènlièA technology that combines multiple disk drives for performance, redundancy, or both.

Unit 10: Boolean Logic & Advanced Theory
English中文PinyinDefinition
Boolean Algebra布尔代数Bù'ěr dàishùA branch of algebra dealing with binary variables and logical operations (AND, OR, NOT).
AND Gate与门Yǔ ménA logic gate that outputs 1 only if all inputs are 1.
OR Gate或门Huò ménA logic gate that outputs 1 if at least one input is 1.
NOT Gate非门Fēi ménA logic gate that inverts the input: outputs 1 if input is 0, and vice versa.
NAND Gate与非门Yǔ fēi ménA logic gate that outputs 0 only if all inputs are 1 (NOT AND).
NOR Gate或非门Huò fēi ménA logic gate that outputs 1 only if all inputs are 0 (NOT OR).
XOR Gate异或门Yì huò ménA logic gate that outputs 1 if the inputs are different.
Logic Circuit逻辑电路Luójí diànlùA physical arrangement of logic gates that performs a Boolean function.
Half Adder半加器Bàn jiā qìA circuit that adds two single bits, producing a sum and a carry.
Full Adder全加器Quán jiā qìA circuit that adds three bits (two inputs and a carry-in), producing a sum and a carry-out.
⭐ Simplification简化JiǎnhuàThe process of reducing a Boolean expression to its simplest form using algebraic rules or Karnaugh maps.
⭐ Minimisation最小化Zuì xiǎo huàFinding the simplest logic circuit that implements a given Boolean function, reducing cost and delay.
⭐ Don't Care Condition无关条件Wúguān tiáojiànAn input combination that will never occur, which can be treated as 0 or 1 to simplify the expression.

⭐ Second Normal Form (2NF)第二范式Dì èr fànshìA table in 1NF where all non-key attributes are fully dependent on the primary key.
⭐ Third Normal Form (3NF)第三范式Dì sān fànshìA table in 2NF with no transitive dependencies (non-key attributes depend only on the key).
