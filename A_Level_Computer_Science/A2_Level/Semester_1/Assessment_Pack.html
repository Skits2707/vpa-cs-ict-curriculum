Quiz 1: Boolean Algebra & Floating-Point (20 Marks, 25 Minutes)

Topic Coverage: Boolean laws, De Morgan's, Karnaugh maps, IEEE 754 floating-point representation, normalisation, error calculation.

Question 1 — Boolean Simplification (6 marks)

1.1 Simplify the following Boolean expression using algebraic laws. Show each step clearly.

F = A·B + A·B' + A'·B

(3 marks)

1.2 Use De Morgan's laws to simplify the expression below. Show your working.

F = (A + B)' · (A · B)'

(3 marks)

Question 2 — Karnaugh Map (6 marks)

2.1 A Boolean function has the following truth table:

Construct a 3-variable Karnaugh map and derive the minimal Sum-of-Products expression. (4 marks)

2.2 Draw the logic circuit for your simplified expression using AND, OR, and NOT gates. (2 marks)

Question 3 — Floating-Point (5 marks)

3.1 Convert the denary number +6.5 to IEEE 754 single-precision floating-point format. Show all steps: binary conversion, normalisation, exponent with bias, and final 32-bit pattern. (4 marks)

3.2 A floating-point representation uses 8 bits for the mantissa and 4 bits for the exponent (both two's complement). State the smallest positive normalised number that can be represented. (1 mark)

Question 4 — Error Calculation (3 marks)

A number is stored as 0.1010 × 2³ (4-bit mantissa). The exact value is 5.125.

4.1 Calculate the absolute error. (1 mark)

4.2 Calculate the relative error as a percentage. (2 marks)

Mark Scheme — Quiz 1: Boolean & Floating-Point

Q1 Boolean Simplification (6 marks)

  • 1.1 F = A·B + A·B' + A'·B = A·(B + B') + A'·B = A·1 + A'·B = A + A'·B = A + B (1 mark per correct step, max 3)
  • 1.2 F = (A + B)' · (A · B)' = A'·B' · (A' + B') = A'·B'·A' + A'·B'·B' = A'·B' + A'·B' = A'·B' (1 mark per correct step, max 3)

Q2 Karnaugh Map (6 marks)

  • 2.1 Correct K-map populated (2 marks); correct grouping (1 mark); minimal SOP: F = B + C (1 mark)
  • 2.2 OR gate with B and C as inputs (2 marks)

Q3 Floating-Point (5 marks)

  • 3.1 6.5 = 110.1₂ = 1.101 × 2² (1 mark); sign = 0, exponent = 2 + 127 = 129 = 10000001₂ (1 mark); mantissa = 10100000000000000000000 (1 mark); final pattern assembled (1 mark)
  • 3.2 0.10000000 × 2⁻⁷ (or equivalent) (1 mark)

Q4 Error (3 marks)

  • 4.1 Stored value = 0.1010 × 2³ = 5.0; Absolute error = |5.125 − 5.0| = 0.125 (1 mark)
  • 4.2 Relative error = 0.125 / 5.125 = 0.02439... ≈ 2.44% (2 marks)
