1. Course Summary and Philosophy

This semester establishes the theoretical foundations of computer science essential for success at A-Level. Building upon the strong IGCSE Computer Science (0478) foundation already established at VPA, students will deepen their understanding of how computers represent information, communicate across networks, and process data at the hardware level. The semester introduces programming fundamentals that will be developed intensively in Semester 2.

教学理念 (Jiàoxué lǐniàn): We employ a bilingual scaffolding approach where core concepts are introduced in English with Mandarin annotations, gradually reducing L1 support as students build academic confidence. Every lesson incorporates EAL strategies including vocabulary pre-teaching, visual anchors, and structured speaking frames.

5. Standard Lesson Structure (40 Minutes)

0–5 min: Starter / Do Now
• Retrieval practice question from previous lesson
• Vocabulary review (bilingual flashcards)
• Quick diagnostic MCQ

5–25 min: Main Activity
• Concept introduction with visual model
• Worked example (teacher modelling)
• Guided practice (pairs/groups)
• Independent practice with Bronze/Silver/Gold tasks

25–35 min: Consolidation
• Plenary check (exit ticket or whiteboard response)
• Common misconception addressed
• Preview of next lesson

35–40 min: Set Homework
• Exam-style question (differentiated)
• Vocabulary consolidation
• Reading/preparation for next lesson

10. Pseudocode Reference (Preview for Semester 2)

CIE Pseudocode Conventions (9618):

DECLARE <variable> : <data_type>
CONSTANT <name> = <value>
INPUT <variable>
OUTPUT <value>
IF <condition> THEN ... ELSE ... ENDIF
FOR <variable> = <start> TO <end> ... ENDFOR
WHILE <condition> DO ... ENDWHILE
REPEAT ... UNTIL <condition>
PROCEDURE <name>(<parameters>) ... ENDPROCEDURE
FUNCTION <name>(<parameters>) RETURNS <type> ... ENDFUNCTION

These constructs will be taught intensively in Semester 2.
